# TSA Webmaster Competition (HS)

Team members: Justin Phu and Christopher Locascio

This website was created with the intent to accomplish the challenge listed on https://tsaweb.org/competitions-programs/tsa/themes-problems:

> "Develop a website that compiles information for homeowners on green and clean energy home solutions and other initiatives. The website should provide basic information, cost, tax rebate information, and clean/green energy practices."

Live Link: https://s0lati.github.io/
